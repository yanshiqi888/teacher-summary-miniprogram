# 教师工作总结小程序 - 使用说明

## 项目简介
这是一个帮助教师快速生成工作总结和计划的微信小程序，支持三种文档类型：
- 班主任学期工作总结
- 科室学期工作总结
- 科室学期工作计划

## 功能特点
✅ 表单填写，智能生成
✅ 历史记录保存
✅ 内容复制导出
✅ 简洁易用的界面

## 如何使用

### 1. 导入项目
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `teacher-summary-miniprogram` 文件夹
4. 使用测试号 AppID 或自己的 AppID

### 2. 生成文档
1. 在首页选择需要的功能
2. 填写表单信息（带 * 为必填项）
3. 点击"生成"按钮
4. 查看生成的文档

### 3. 管理历史记录
- 点击"历史记录"查看所有生成的文档
- 可以重新查看、删除单条或清空全部

### 4. 导出文档
- 在预览页面点击"复制内容"
- 粘贴到 Word 文档中保存

## 当前版本说明
这是 **Mock 版本**，使用预设模板生成文档。

### Mock 版本特点
- ✅ 无需后端服务器
- ✅ 无需 API key
- ✅ 完全免费
- ✅ 响应速度快
- ⚠️ 生成内容基于固定模板

### 升级到 AI 版本
如果需要更智能的内容生成，可以：
1. 搭建后端服务（Node.js + Express）
2. 接入 AI API（OpenAI / 文心一言 / 通义千问等）
3. 修改生成逻辑调用真实 AI 接口

## 导出 Word 功能
当前版本支持"复制内容"，手动粘贴到 Word。

如需真实的 Word 导出功能，需要：
1. 搭建后端服务
2. 使用 `docxtemplater` 或 `officegen` 等库生成 Word
3. 返回下载链接给小程序

## 技术栈
- 微信小程序原生开发
- 本地存储（wx.setStorageSync）
- 模板字符串生成文档
